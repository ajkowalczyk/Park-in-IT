# 🅿️ Park It!

A 2D parking simulation / puzzle game built in Python.

The goal is simple… at least in theory:
Park your car perfectly without hitting obstacles, other vehicles, or your own patience limit.

While the implementation and architecture are my own, the core gameplay concept is inspired by the classic title *Parking Mania*.

Tbh just wanted to do something fun :D 

Yes, i used AI to drop this readme and only added few lines of mine, won't lie. *Sprites are probably AI-Generated.* **Code is not.**

---

## 🎮 Features (Planned more than In Progress)

* 🚗 Player-controlled car 
* 🧱 Obstacles and tight parking spaces
* 🅿️ Precision parking detection
* 🗺️ Multiple levels / parking maps
* 🔊 Sound effects & engine audio
* 💾 Save system
* ⏱️ Timers and scoring system
* 🤖 NPC traffic
